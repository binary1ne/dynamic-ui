# Angular CLI Frontend (Enterprise AI Platform)

This project was generated with Angular CLI version 17.

## Development server

Run `npm start` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Build

Run `npm run build` to build the project. The build artifacts will be stored in the `dist/` directory.

## Backend Connection

Make sure the Flask backend is running at `http://localhost:5000` before starting the Angular app.

See the main `README.md` in the project root for complete setup instructions.

## Environment

- Development API URL: `http://localhost:5000/api`
- Production: Update in `src/environments/environment.prod.ts`

## Features

- Modern Tailwind CSS styling
- JWT authentication with HTTP interceptors
- Role-based routing with guards
- Lazy loading for admin components
- Responsive design
